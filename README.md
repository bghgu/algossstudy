## 진행방식

1. 매주 목요일에 모여서 스터디를 진행한다.
2. 리더가 정해주는 문제 + a 를 각자 풀어서 커밋한다.
3. 매주 두 사람씩 돌아가면서 원하는 알고리즘을 실제로 구현가능할 정도로 설명한다.
4. 원하는 언어로 문제를 풀되 한가지 언어로 꾸준히 푸는 연습을 한다.
